# imgcolor

## 提取图片主色

>1.透明颜色显示为#ffffff<br />
>2.图片过大会造成卡顿，请谨慎使用

## 图片转ascii

>1.无法生成分辨率过大的图片<br />
>2.线程阻塞无法停止<br />
>3.图片过大会造成卡顿
